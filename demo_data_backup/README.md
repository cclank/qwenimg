# QwenImg 示例数据

这个目录包含了 QwenImg 的示例数据，用于在新环境中快速展示应用效果。

## 📦 内容

- `outputs/` - 5 张真实生成的示例图片
  - 🐕 柴犬在樱花树下
  - 🐴 奔跑的白色骏马
  - 🐶 可爱的小狗
  - 🐱 卡通小黑猫
  - 🐱 可爱的小橘猫
- `sample_tasks.sql` - 数据库记录（包含提示词和任务信息）
- `restore.sh` - 一键恢复脚本

## 🚀 使用方法

### 自动恢复（推荐）

运行 `./install.sh` 时会自动检测并恢复示例数据，无需手动操作。

### 手动恢复

如果需要单独恢复示例数据：

```bash
cd demo_data_backup
./restore.sh
```

## 📝 注意事项

- ✅ 示例数据会在安装时自动恢复
- ✅ 使用独立的会话 ID，不会与你的实际数据混淆
- ✅ 只在首次安装时恢复，不会覆盖已有数据
- ✅ 可以随时删除这些示例数据，不影响应用使用

## 🔄 重新生成示例数据

如果你想用自己的图片替换示例数据：

```bash
# 在项目根目录运行
./backup_demo_data.sh
```

这会备份当前数据库中最新的 5 张图片作为新的示例数据。
