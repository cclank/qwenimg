# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-XX

### Added
- Initial release of QwenImg
- Support for text-to-image generation (wan2.5-t2i-preview)
- Support for image-to-video generation (wan2.5-i2v-preview)
- Support for text-to-video generation (wan2.5-t2v-preview)
- Automatic API key detection from environment variables and .env file
- PIL.Image object support for easy image manipulation
- Multiple image input formats (local file, URL, Base64)
- Comprehensive examples and documentation
- Region support (Beijing and Singapore)
- Customizable parameters (size, resolution, duration, seed, etc.)
